# Weather Dashboard Web App

Web application for displaying real-time weather data with a simple login system and user dashboard. Users can sign in, search for a city, and view current weather information using the OpenWeather API.

## Live Demo
https://simpleweathersite.onrender.com

## Tech Stack
- JavaScript
- HTML5
- CSS3
- Node.js
- OpenWeather API

## Features
- User authentication (login system)
- User dashboard
- Real-time weather data
- City search
- Weather forecast
- User profile pictures
- Logout functionality

## Installation and Local Run

1. Clone the repository:
git clone https://github.com/Darkillexxx/webserver_test.git

2. Go to the project folder:
cd webserver_test

3. Install dependencies:
npm install

4. Start the server:
node app.js

5. Open in browser:
http://localhost:3000
